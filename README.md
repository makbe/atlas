# Project Atlas

**Project Atlas** — исследовательская и продуктовая база знаний для создания системы развития человеческого потенциала.

## Как открыть
1. Установите Obsidian.
2. Выберите **Open folder as vault**.
3. Укажите папку `project-atlas`.
4. Откройте `HOME.md`.

## Как подключить Git
```bash
cd project-atlas
git init
git add .
git commit -m "Initialize Project Atlas knowledge base"
git branch -M main
git remote add origin <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
git push -u origin main
```
