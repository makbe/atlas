# Правила ведения Project Atlas

## Создание документа
1. Выберите раздел и префикс.
2. Возьмите следующий свободный номер.
3. Скопируйте шаблон из `90_templates`.
4. Назовите файл: `ID-краткое-название.md`.
5. Добавьте ссылку в `INDEX.md` раздела.

## Статусы
`planned`, `draft`, `review`, `accepted`, `superseded`, `archived`.

## Коммиты Git
```text
type(scope): краткое описание
```
Примеры:
```text
docs(constitution): add data ownership principle
research(potential): add literature review
decision(product): approve first MVP audience
fix(index): repair broken links
```

## Разделение знаний
Всегда различайте установленный факт, интерпретацию, гипотезу, решение и открытый вопрос.
